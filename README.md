
# 🌤️ Weather App

A simple and responsive **Weather Application** built using **Node.js, Express.js, EJS, HTML, CSS, and JavaScript**. The application fetches real-time weather information using the **OpenWeather API** based on the **latitude and longitude** entered by the user.

🌐 **Live Demo:** https://weather-app-nbvb.onrender.com

## ✨ Features

- 🌍 Get current weather information using latitude and longitude.
- 🌡️ Displays current temperature.
- 🤒 Shows "Feels Like" temperature.
- 💧 Displays humidity.
- 🌬️ Shows wind speed.
- 👀 Displays visibility.
- 📈 Shows atmospheric pressure.
- 🌅 Displays sunrise time.
- 🌇 Displays sunset time.
- 🕒 Shows the last updated time.
- ☁️ Displays weather condition along with an official weather icon.
- 🎨 Clean and responsive user interface.
